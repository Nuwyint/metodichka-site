# Инструкция по извлечению изображений из PDF

## Автоматический способ (требует установки зависимостей)

1. Установите зависимости:
```bash
npm install pdfjs-dist canvas
```

2. Поместите PDF файл в корень проекта с именем:
   `ПРОЕКТИРОВАНИЕ И РАЗРАБОТКА МУЛЬТИМЕДИЙНОГО КОНТЕНТА И ПОЛЬЗОВАТЕЛЬСКОГО ИНТЕРФЕЙСА.pdf`

3. Запустите скрипт:
```bash
node extract-pdf-images.js
```

Или укажите путь к PDF:
```bash
node extract-pdf-images.js "путь/к/файлу.pdf"
```

## Ручной способ (рекомендуется)

### Вариант 1: Adobe Acrobat
1. Откройте PDF в Adobe Acrobat
2. Инструменты → Редактирование PDF → Экспорт → Изображение
3. Выберите формат PNG или JPG
4. Сохраните изображения в папку `public/media/`

### Вариант 2: Онлайн-инструменты
1. Используйте онлайн-сервисы:
   - https://www.ilovepdf.com/pdf-to-jpg
   - https://smallpdf.com/pdf-to-jpg
   - https://www.adobe.com/acrobat/online/pdf-to-jpg.html
2. Загрузите PDF
3. Экспортируйте все страницы как изображения
4. Сохраните нужные изображения в `public/media/`

### Вариант 3: Скриншоты
1. Откройте PDF в любом просмотрщике
2. Найдите нужные страницы с рисунками
3. Сделайте скриншоты нужных частей
4. Сохраните в `public/media/` с правильными именами
